# アプリ名Myeatーhukuoka
 ## 福岡の美味しい食べ物、お店を気軽に探せる投稿することができます
 [Myeat-hukuoka](http://3.114.131.61/ "Myeat-hukuoka")

  ID genta

  投稿者アカウント

  ##### 利用方法
  新規登録することで自分の好きなお店や食べ物を投稿することができます
  登録しなくても投稿の内容は見れます。
  ###### 制作背景
  自分が静岡から引っ越してきてから美味しいお店を知りたい地元民ならではのいろんなお店を知ることができるアプリがあればいいなと思ったのと、このコロナ環境で自分のお店の宣伝を気軽にできるようにと考えました。
